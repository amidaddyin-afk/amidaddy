## ‚ùì Troubleshooting

### "fatal: remote origin already exists"
```powershell
git remote remove origin
# Then run the git remote add command again
```

### "Permission denied (publickey)"
```powershell
# You need a GitHub token instead of password
# Follow the credentials section above
```

### "Branch 'master' doesn't exist"
```powershell
# Just run:
git branch -M main
git push -u origin main
```

### "Nothing happens when I push"
```powershell
# Check status:
git status

# Check remotes:
git remote -v

# Check log:
git log --oneline -3
```

### Still stuck?
1. Read [GITHUB_SETUP.md](./GITHUB_SETUP.md) for detailed guidance
2. Check GitHub's help: https://docs.github.com
